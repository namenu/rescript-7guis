# ReScript 7GUIs

[7GUIs](https://eugenkiss.github.io/7guis/) tasks implemented in [ReScript](https://rescript-lang.org/).

## Usage

```bash
# install depdendencies at first
yarn install

# build rescript
yarn build

# run web server
yarn serve

# then open in the browser
# open http://localhost:3000
```
